## 2. Giới thiệu website

**Gemstone** là cửa hàng chuyên cung cấp các sản phẩm trang sức phong thủy được chọn lọc kỹ càng, dành cho những khách hàng yêu thích vẻ đẹp tự nhiên và ý nghĩa tâm linh.

Website cung cấp đa dạng các dòng trang sức phong thủy như: **vòng tay, nhẫn, mặt dây chuyền, charm và chuỗi hạt**. Mỗi sản phẩm đều được chọn lọc để cân bằng giữa giá trị thẩm mỹ và ý nghĩa tâm linh, đại diện cho các giá trị: **tài lộc – bình an – may mắn – sức khỏe**.

Thay vì hướng tới sự phức tạp, Gemstone đề cao phong cách **sang trọng, gần gũi và bền vững**. Doanh nghiệp tập trung vào:

- Tư vấn cá nhân hóa, giúp khách hàng tìm được sản phẩm phù hợp với nhu cầu và thẩm mỹ riêng
- Minh bạch về ý nghĩa sản phẩm và giá trị phong thủy
- Lấy sự hài lòng của khách hàng làm cam kết hàng đầu

Gemstone hướng đến việc cung cấp những sản phẩm vừa là phụ kiện thời trang nâng tầm phong cách cá nhân, vừa mang ý nghĩa tích cực cho người đeo và người nhận quà.

## 7. Hướng dẫn cài đặt

### Yêu cầu hệ thống

- PHP >= 7.4 (khuyến nghị 8.x)
- MySQL >= 5.7 hoặc MariaDB >= 10.3
- Web server: Apache hoặc Nginx
- Công cụ: XAMPP / WAMP / Laragon (local) hoặc hosting thật

---

### Bước 1 — Cài đặt WordPress

1. Tải WordPress tại [https://wordpress.org/download/](https://wordpress.org/download/)
2. Giải nén và copy toàn bộ thư mục vào thư mục gốc web server:
   - **XAMPP**: `C:/xampp/htdocs/gemstone/`
   - **Laragon**: `C:/laragon/www/gemstone/`
3. Tạo database mới trong **phpMyAdmin** (ví dụ: `gemstone_db`)
4. Truy cập `http://localhost/gemstone` trên trình duyệt
5. Làm theo hướng dẫn cài đặt WordPress:
   - Nhập thông tin database: tên DB, username, password, host (`localhost`)
   - Đặt tên site, tài khoản admin và email
6. Hoàn tất cài đặt — đăng nhập vào trang quản trị tại `http://localhost/gemstone/wp-admin`

---

### Bước 2 — Cài đặt và kích hoạt plugin All-in-One WP Migration

1. Đăng nhập vào **WordPress Admin** (`/wp-admin`)
2. Vào menu **Plugins → Add New Plugin → Upload Plugin**
3. Nhấn **Choose File**, chọn file `all-in-one-wp-migration.7.68.zip` → nhấn **Install Now** → **Activate Plugin**
4. Tiếp tục upload file `all-in-one-wp-migration-unlimited-extension.zip` theo cách tương tự → **Install Now** → **Activate Plugin**
5. Plugin xuất hiện trên menu trái với tên **All-in-One WP Migration**

> **Lưu ý:** Cả 2 file plugin `.zip` có trong thư mục source code của project.

---

### Bước 3 — Import dữ liệu website Gemstone

1. Giải nén file backup:
   ```
   gemstone.sharecs.net-20260516-075058-ez6kyq.rar
   ```
   → Thu được file: `gemstone.sharecs.net-20260516-075058-ez6kyq.wpress`

   > **Lưu ý:** Nếu không giải nén được, tải file `.rar` tại link:
   > [https://gemstone.sharecs.net/wp-content/uploads/2026/05/gemstone.sharecs.net-20260516-075058-ez6kyq.rar](https://gemstone.sharecs.net/wp-content/uploads/2026/05/gemstone.sharecs.net-20260516-075058-ez6kyq.rar)

2. Vào **WordPress Admin → All-in-One WP Migration → Import**

3. Nhấn **Import From → File**, chọn file `.wpress` vừa giải nén

4. Chờ quá trình upload và import hoàn tất (có thể mất vài phút tùy dung lượng)

5. Khi xuất hiện thông báo **"Import process is complete"**, nhấn **Finish**

6. WordPress sẽ tự động đăng xuất — đăng nhập lại bằng tài khoản demo bên dưới

> **Lưu ý quan trọng:**
> - Không tắt trình duyệt hoặc server trong quá trình import
> - Sau khi import, toàn bộ dữ liệu (sản phẩm, trang, cài đặt, theme, plugin) sẽ được khôi phục theo đúng bản gốc
> - Nếu gặp lỗi giới hạn dung lượng upload, tăng `upload_max_filesize` và `post_max_size` trong `php.ini`

## 8. Hướng dẫn chạy project

### Chạy trên môi trường Local (XAMPP / Laragon)

1. Khởi động web server và MySQL:
   - **XAMPP**: Mở XAMPP Control Panel → Start **Apache** và **MySQL**
   - **Laragon**: Nhấn **Start All**

2. Đảm bảo WordPress đã được cài đặt và import dữ liệu xong (xem [Bước 3](#bước-3--import-dữ-liệu-website-gemstone))

3. Truy cập website tại trình duyệt:
   ```
   http://localhost/gemstone/
   ```

4. Truy cập trang quản trị:
   ```
   http://localhost/gemstone/wp-admin
   ```

5. Đăng nhập bằng [tài khoản demo](#9-tài-khoản-demo) để quản lý nội dung

### Chạy trên Hosting

1. Upload toàn bộ file WordPress lên hosting qua **FTP** (FileZilla) hoặc **File Manager** của hosting
2. Tạo database MySQL trên **cPanel** hoặc panel quản lý hosting
3. Cập nhật thông tin kết nối DB trong file `wp-config.php`:
   ```php
   define( 'DB_NAME', 'tên_database' );
   define( 'DB_USER', 'username_db' );
   define( 'DB_PASSWORD', 'password_db' );
   define( 'DB_HOST', 'localhost' );
   ```
4. Thực hiện import file `.wpress` theo [Bước 3](#bước-3--import-dữ-liệu-website-gemstone)
5. Trỏ domain về thư mục chứa WordPress
6. Truy cập domain để kiểm tra
